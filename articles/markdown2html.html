2026-01-04 随笔首发

为什么写博客？

建立个人博客是一个技术人的基本修养，它能帮助你：

整理知识

"写作是更好的思考" - 每写一篇文章，都是对知识的深度梳理。

分享价值

把踩过的坑、学到的技巧分享出来，帮助他人的同时也在巩固自己的理解。

建立品牌

一个持续更新的技术博客是程序员最好的简历。

这个博客的特点

  1. 极简设计 - 不追求花哨，专注内容
  2. 静态部署 - 安全、快速、无需服务器端
  3. Markdown写作 - 用最舒服的方式写文章

快速开始

# 1. 克隆项目
git clone https://github.com/your/blog.git

# 2. 安装依赖
npm install

# 3. 写文章
# 在 drafts 文件夹创建 .md 文件

# 4. 发布
npm run publish

写在最后

感谢访问，希望这个博客对你有所帮助！
